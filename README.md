# Smart Code Reviewer ⚡

An elegant, high-performance, full-stack AI-driven web platform that evaluates script blocks, checks for logic vulnerabilities, and outputs clean markdown summaries and downloadable PDF audits. Powered by Groq for lightning-fast inference.

## 🚀 Tech Stack

- **Frontend:** React, Vite, Monaco Code Editor, Axios, React Markdown, jsPDF
- **Backend:** Node.js, Express, MongoDB
- **AI Core Engine:** Groq Cloud API (Llama 3 / Mixtral)
